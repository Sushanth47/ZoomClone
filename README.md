# ZoomClone

#To initialize our app
$ npm init

#To Install Express to app
$ npm install express

#To install Socket.IO
$ npm install socket.io

#To install peer js
$ npm install peerjs

#To install UUID
$ npm install uuid

#To Install BootStrap
$ npm install bootstrap

#To install Google APIs
$ npm install googleapis



If any error occurs try 
$ npm audit fix
